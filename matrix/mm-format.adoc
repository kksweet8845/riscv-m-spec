Format for Matrix Mask Instructions under OP-Xmaze1


// OPZFX
```wavedrom
{reg:[
  {bits: 7, name: 0x6b, attr: 'OPZMX'},
  {bits: 5, name: 'Md', attr: ['destination'], type: 2},
  {bits: 3, name: 0x0},
  {bits: 5, name: 'Ms1', attr: 'source 1', type: 4},
  {bits: 5, name: 'Ms2', attr: 'source 2', type: 4},
  {bits: 1, name: 'vm'},
  {bits: 6, name: 'funct6'}
]}
```
